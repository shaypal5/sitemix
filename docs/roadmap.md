# Roadmap

## MVP (v0.1)

- Working `page` and `site` commands.
- Trafilatura-first extraction with short-text fallback attempts.
- Sitemap support (`xml` and text list) + BFS fallback discovery.
- Markdown/JSON/XML deterministic output.
- Basic tests, CI, and packaging.

## Post-MVP (v0.2+)

- Better subtitle/date/language enrichment.
- Optional cache and resume support.
- Better sitemap index recursion and large sitemap chunk support.
- Optional dedupe by content hash.
- Structured telemetry and richer run stats.
- Extended tests with recorded fixtures.
